## 项目介绍 
这是一个简单的用户管理系统，用于管理用户的基本信息和状态。

## 功能介绍
- 用户注册：用户可以通过注册页面填写个人信息进行注册。
- 用户登录：已注册用户可以使用自己的账号和密码登录系统。
- 用户信息管理：登录后，用户可以查看和修改自己的个人信息，如用户名、年龄、邮箱等。
- 用户状态管理：系统管理员可以查看所有用户的状态，并对用户进行激活或禁用操作。

## 快速开始 
1. 安装依赖：在项目根目录下运行 `npm install` 安装项目依赖。
2. 启动项目：运行 `npm start` 启动项目。
3. 访问项目：在浏览器中访问 `http://localhost:3000` 即可打开项目。
