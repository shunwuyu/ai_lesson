# VS Code Markdown 微信公众号预览插件 - 任务拆分文档

## 1. 拆分原则

- **优先级驱动**：优先完成高优先级需求
- **可交付性**：每个阶段完成后可展示可见效果
- **依赖清晰**：明确任务间的前置依赖
- **粒度适中**：单个任务可在 1-2 小时内完成

## 2. 总体任务概览

| 阶段 | 任务数量 | 预计时间 | 可见效果 |
|------|----------|----------|----------|
| 一、项目基础搭建 | 5 | ~2h | 项目可编译运行 |
| 二、命令注册与入口 | 4 | ~2h | 命令可执行 |
| 三、WebView 面板创建 | 5 | ~3h | 预览窗口可显示 |
| 四、内容渲染集成 | 6 | ~4h | Markdown 可渲染 |
| 五、实时同步机制 | 4 | ~3h | 编辑实时更新 |
| 六、复制功能 | 3 | ~2h | 一键复制可用 |
| 七、图片粘贴（中优先级） | 5 | ~4h | 图片粘贴可用 |
| 八、测试与优化 | 4 | ~3h | 稳定性提升 |

- 调用 md-wx.renderHtml() 渲染
- 将渲染结果插入 WebView DOM

**产出**：Markdown 内容可渲染为 HTML
