# deepseek-r1-webgpu

今天我们给大家分享一个，可以写在简历中的超然项目， 它覆盖以下技能点：

- 端模型
不同于传统的大模型api 调用， 它是一个基于 webgpu 的模型， 它可以在浏览器中运行， 且可以支持多线程。

端侧本地模型对比云端 LLM API 核心价值突出：数据全程本地运算不外传，隐私安全更强；支持离线使用，不受网络制约；无持续 Token 计费，长期成本更低（ollama）；省去网络往返，推理延迟更小。同时不受服务商限流、调价、下线限制，可自主定制量化与推理逻辑，适配浏览器、移动端等各类终端，是离线、隐私类 AI 应用最优方案。

- tailwindcss 

Tailwind CSS 是实用优先的原子化 CSS 框架，内置大量预设工具类，无需手写自定义样式，直接组合类名快速搭建页面。无需切换样式文件，高度可配置，体积可控，适配前端各类项目，大幅提升开发效率。

原子类语义直观、低歧义，AI 易精准生成修改， 适配 AI 快速生成 UI 全流程。

- react + typescript

大型 AI 前端项目更稳定易迭代， 代码可读性与可维护性大幅提升。

- webworkers
Web Workers 是浏览器多线程方案，允许 JS 在后台独立线程运算，避免复杂计算阻塞页面 UI 渲染。
