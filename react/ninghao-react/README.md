    - 避免全局安装
        使用 npx 执行包内的命令行工具时，不需要将这些工具全局安装到你的系统上。这有助于减少全局安装的依赖数量，避免版本冲突，以及保持环境的清洁。
    - 临时安装与执行    
        当你运行一个命令时，npx 会临时下载并安装所需的包（如果本地没有），然后直接执行。执行完毕后，这些临时安装的包通常会被清理，减少了系统残留。
