- XSS 例子
XSS 全称是 Cross-Site Scripting，中文叫 跨站脚本攻击。
简单说就是黑客把恶意的 JavaScript 代码注入到网页里。比如一个搜索框，用户输入内容后直接显示在页面上，如果网站没做处理，黑客就可以输一段脚本，比如 <img src=x onerror=alert('xss')>，当页面渲染时，这段代码就会被执行。轻则弹个窗，重则偷 Cookie、劫持会话、钓鱼登录，甚至控制用户账号。XSS 分反射型、存储型和 DOM 型，核心问题都是没对用户输入做转义或过滤。防御的关键就是：别信用户输入，输出前一定要编码，或者用 textContent、安全库来处理。
