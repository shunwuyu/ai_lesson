- 何为diff算法?
- 对比两棵 vdom 树，找到有差异的部分的算法，就叫做 diff 算法
- 两模树作diff 复杂度是O(n^3)
    - 每个节点对比另一棵树的全部节点 n
    - 插入、删除、修改 n
    - 所有节点   n
    1000   10亿次

- 怎么解决时间复杂度？
    - 只做同层的对比， type变了就不再对比子节点
    跨层级比较少， 没必要  一般都在同一层级
