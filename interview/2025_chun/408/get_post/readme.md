| 特性 | GET | POST |
|------|-----|------|
| 数据位置 | URL参数中 | 请求体中 |
| 数据大小 | 受URL长度限制（约2KB） | 无限制 |
| 安全性 | 参数暴露在URL中 | 数据在请求体中相对安全 |
| 缓存 | 可被缓存 | 默认不缓存 |
| 浏览器历史 | 参数保留在历史记录 | 不会保留在历史记录 |
| 书签收藏 | 可带参数收藏 | 不可带参数收藏 |
| 数据类型 | 只允许ASCII字符 | 支持任意数据类型 |
| 幂等性 | 幂等（多次请求结果相同） | 非幂等 |
| 适用场景 | 数据查询、筛选 | 数据提交、上传文件 |
