# 数据库 创建连接

1. 配置连接参数
使用 SQLAlchemy 连接数据库，是通过设置一个固定格式的字符串来实现的。以 mysql+aiomysql 为例，那么其连接格式如下：

DB_URL="mysql+aiomysql:// 用户名：密码 @主机名：端口号 / 数据库名称？charset=utf8mb4"

DB_URL="mysql+aiomysql://root:root@127.0.0.1:3306/book_db?charset=utf8mb4"

2. 创建 Engine 对象
SQLAlchemy 中的 Engine 对象，它负责管理数据库连接的创建（并不直接操作数据库）、连接池的维护、SQL 语句的翻译等。Engine 对象在整个程序中只能有一个。创建 Engine 对象代码如下：

CREATE DATABASE sqlalchemy_demo CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
