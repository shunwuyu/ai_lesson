Agent 能力 API

四大核心能力

Anthropic API 推出四种新功能，使开发者能够构建更强大的 AI 代理：代码执行工具、MCP 连接器、Files API 和最多一小时的提示缓存能力。

无需构建自定义基础设施

结合 Claude Opus 4 和 Sonnet 4，这些 beta 功能使开发者能够构建执行高级数据分析代码、连接外部系统的代理。

MCP 连接器

使开发者能够将 Claude 连接到任何远程模型上下文协议 (MCP) 服务器，无需编写客户端代码。

自动处理连接管理、工具发现和错误处理

只需在 API 请求中添加远程 MCP 服务器 URL，即可立即访问强大的第三方工具，显著降低了构建启用工具代理的复杂性。

文件 API 与扩展缓存

Files API 简化了开发者在使用 Claude 构建时存储和访问文档的方式，可上传文档一次并在对话中反复引用。

提示缓存最长可达 1 小时，比标准的 5 分钟提高了 12 倍

扩展缓存可将长提示的成本降低高达 90%，延迟降低高达 85%，使构建能在较长时间内保持上下文的代理成为可能。

代码执行工具

沙箱环境执行

Anthropic API 上的代码执行工具使 Claude 能够在沙箱环境中运行 Python 代码，生成计算结果和数据可视化。

将 Claude 从代码编写助手转变为数据分析师

能够加载数据集、生成探索性图表、识别模式，并根据执行结果迭代改进输出，全部在单次交互中完成。

核心应用场景

金融建模：生成财务预测、分析投资组合和计算复杂财务指标

科学计算：执行模拟、处理实验数据和分析研究数据集

商业智能：创建自动报告、分析销售数据和生成绩效仪表板

文档处理：跨格式提取和转换数据、生成格式化报告

统计分析：执行回归分析、假设检验和预测建模

与 Files API 集成

Files API 将与代码执行工具集成，使 Claude 能够在代码执行期间直接访问和处理上传的文件，并生成图表等文件作为响应的一部分。

每天免费使用 50 小时

组织每天可免费使用代码执行工具 50 小时，超出部分按每容器每小时 $0.05 收费。
