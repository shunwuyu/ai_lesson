## karaoke-style teleprompter

视频录制工具，它是一个提词器，就是让你录制视频的时候，可以看到逐字稿。
传统的提词器是按固定的速度网上滚的， 有时候说快了或慢了，他就跟不上了。自适应说话速度的提词器。

https://x.com/zarazhangrui/status/1992430486539702368


Make a video recording tool with an adaptive teleprompter. Allow the user to connect to their camera to record a video. 

Allow user to paste in a script, and display the script as overlay text during recording like a teleprompter. Use automatic speech recognition to listen to the user's speech in real time, and highlight the words the user already said in real time, in karaoke style. The speed of the teleprompter movement should be in sync with the user's actual speech. So it would only move to the next line when the ASR system detects the user has finished saying the current line. 

After recording, allow user to export video as an MP4 file.

制作一个带有自适应提词器的视频录制工具。允许用户连接摄像头进行视频录制。

用户可以粘贴一段脚本，录制时该脚本会以叠加文字的形式显示在画面上，就像提词器一样，词和词之间要保持一定的间距。

利用自动语音识别（ASR）技术实时监听用户的语音，并以“卡拉OK”式高亮方式实时标出用户已经说出的词语。

提词器的滚动速度应与用户实际语速同步——只有当ASR系统检测到用户已说完当前行的内容后，才会自动滚动到下一行。

录制完成后，允许用户将视频导出为 MP4 文件。
