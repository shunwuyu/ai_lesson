# Ai 多角色配音软件

- 做视频时，有个问题。

市面上多角色配音软件我试了一圈，没一个趁手的。

媒小三
多角色 AI 配音，1300 + 音色，自动分角，支持声音克隆与情感调节。

- 操作很繁琐
  - 比如我发现写好的脚本需要复制到配音软件里，配音配完然后再导到剪辑软件里。
  还有一系列的操作， 一句句的导入， 累死累活。一天只能做出一条视频。
  在不同的软件之间反复横跳，整个人都不好了
  - 有些软件国内用不了，还封号，订阅费好几百。
  - 可控性太差， 好不容易配完音了， 有一句话语速太快了， 调整语速需要重新生成整段，
  重新消耗token，费钱。
  - 你**真的**想这样做？
  真的怀疑语气，软件没有办法做到


决定用AI编程的方式造一个工具

## 项目功能

- 改一句话不用总重新整段生成。
- 每个角色可以单独调整三个参数。
  还要能控制重音语气，之后可以独立音轨。

## 安装环境
