## 端到端的聊天机器人

使用claude code 探索代码库

Claude Code 擅长探索代码库，核心在于其自主探索与超大上下文能力。它能像资深工程师一样主动扫描项目，快速梳理出模块间的依赖关系和核心逻辑，而非被动等待指令。

- 如何用这个工具快速熟悉大型代码库
    git clone https://github.com/https-deeplearning-ai/starting-ragchatbot-codebase.git

    - claude 开聊
    - give me an overview of the codebase
        主动在代码库里找， 找出最重要的文件，向我描述这个应用程序的运行机制。

        无需逐个查找每个文件，而是通过智能搜索找到最相关的内容，
        得到架构信息、关键组件、和部分功能介绍
    - how are these documents processed?
    - trace the process of handling a user's query from frontend to backend.
