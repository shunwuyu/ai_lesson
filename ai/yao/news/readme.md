# 用claude Code 加Skills 零代码搭建每日AI新闻速递

每天， 我们都会被海量的AI 信息淹没。 想要了解行业动态，
但没有时间一个一个网站去刷？

让我们用一个命令，一句话，让AI 帮我们自动抓取， 去翻译
去整理每日的一个AI要闻， 同时生成精美的个人日报的网页。

Claude 有一个超强的预制技能 Skill Creator
它是我们创建自定义skill 的一个基建工具。

当你想要去扩展claude的一个能力， 比如说你要去添加专业
知识，你要去自动化你的的工作流， 或者集成第三方的工具

可以用它生成标准化的一个技能包。
```
安装skill creator, 它的项目地址是：https://github.com/anthropics/skills/tree/main/skills/skill-creator
```
安装好后， 可以问 
```
skill-creator 是什么？
```
需求

```
帮我创建一个 skill，下面是详细描述：Skill 名称
tai-ai-daily（邰学姐每日 AI 新闻概览）
核心功能
为你精选每日最值得关注的 AI 行业资讯，让你用最少的时间掌握最新的行业动态。
具体需求
精选资讯抓取
收集近 24 小时内的 AI 领域重要新闻
覆盖权威 AI 科技媒体（TechCrunch、The Verge、Hacker News 等）
智能过滤，只保留有价值的行业资讯
智能摘要生成
将抓取内容转化为中文精华版
每条资讯提炼出 50–100 字核心要点
智能标注关键词标签，方便快速定位感兴趣的内容
清爽界面展示
输出精美的可视化 HTML 页面
顶部显示：「邰学姐 AI 日报 - [日期]」
使用卡片式设计，每条资讯一目了然
每张卡片可直接点击查看原文详情
```
输入 今日AI 日报
claude  自动工作， 它会访问指定的网站， 爬取AI相关内容
去生成中文摘要， 同时整理格式，再过一到两分钟，一份精美的
网页版就呈现出来了。

# 总结

Agent 配置了定制化的Skill, 我们能实现什么呢？

完全自动化的信息筛选和处理
不需要写一行代码，只需要清晰去表达需求，
Agent 就能帮我们从海量的信息中去提取精华，
自动翻译和摘要，去生成可视化的报告。
信息收集的场景， 行业竞品的监控， 邰姐技术博客的跟踪。

Agent + skills 的能力， 把重复信息获取的能力交给AI
让我们真正专注有价值的思考和决策。


skill-creator 为什么这么牛掰呢？

首先， 他定义了创建技能的六步的标准流程
1. 理解需求， 明确你要去解决的是什么问题
2. 规划内容，列出需要的脚本和参考资料
3. 初始化， 会运行init_skill.py
会自动生成项目的框架
4. 编辑完善， 你去填充具体的内容和资源文件
5. 打包验证， 用package_sill.py 去检查格式并且打包
6. 迭代优化， 根据实际使用的效果，去做持续改进

完整的标准化工作流， 像搭积木一样帮我们自定义skill。
