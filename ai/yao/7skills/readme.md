## Talk a little bit More
再聊一会儿
作者因为不太会聊天，干脆做了一个skill, 专门去复盘自己， 为什么会把天聊死。
你把聊天记录丢进去， 它会告诉你，聊天是从那一句变冷的， 当时还可以怎么去接，
它可以帮我们复盘为什么对方回复哈哈， 这段聊天就没有然后了。

帮我安装skill https://github.com/caibucaiAI/talk-a-little-bit-more
