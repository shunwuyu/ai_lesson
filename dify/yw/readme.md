# Dify 开发本地Agent 智能体

当我们纠结怎么写prompt提示词的时候，顶尖的AI开发者已经用Dify搭建数字员工了。不用写任何一行代码，就能让AI读取本地数据库，调用第三方复杂的工具，openclaw小龙虾、hermes 7x24小时不停机的工作。

## Dify

如果你想掌握AI底层控制权、实现更加私密、更加复杂、更具商业价值的应用，那么就用dify来开发企业级的AI Agent。

AI Agent 是能自主思考、规划、调用工具、一步步完成复杂任务的智能体。
