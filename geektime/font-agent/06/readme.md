# 拍照记单词

基于大模型 API 开发 Web 应用。

英语学习类的App 有很多做的很棒的，善于记单词的扇贝，智能间隔重复算法”精准规划复习时间，确保单词在即将遗忘时被强化，从而以最高效率实现长期记忆。

百词斩,以“图背单词”闻名，利用图像记忆法帮助用户快速建立单词与含义的联系，适合视觉型学习者。
![](1.png) 
awkward(尴尬) giraffe(长颈鹿)

生活中使用英语，出国旅游，跨国购物、点餐，结合AIGC 生成特性， 拍照即得单词卡片与情景对话，让用户在真实场景中高效记忆地道表达。

## 核心功能

核心功能只有一个，就是用户通过手机拍一张照片，让 AI 识别出照片主体，找到一个最合适的英文单词，生成一张单词卡片。卡片内容包括单词的拼写、读音、例句，以及生成一段对话，以帮助用户快速记忆这个单词和掌握这个单词的用法。
