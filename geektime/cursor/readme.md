## chrome插件

- 选中页面上的内容， 翻译成制定的语言
- 点击按钮，总结页面的内容
  kimi 

开发一个Chrome插件，具有以下两个功能：
1. 选中页面的内容，然后点击插件的按钮，可以翻译选中的内容。
2. 总结页面的内容。

要求：
1. 翻译功能和总结功能都使用kimi api，请参见Kimi api文档。
2. 打开在侧边栏中。
