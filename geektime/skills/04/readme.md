## PRD 写作的真实痛点

核心问题：很多产品想法在早期阶段非常模糊。你可能只有一个念头：“我想做一个类似瑞幸的积分兑换功能。”但具体怎么做、积分怎么获取、怎么消费、要不要会员等级……这些问题不澄清清楚，PRD 根本没办法写。

PRD就是为了解决这个问题而诞生的。
