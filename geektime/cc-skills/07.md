# AI 日报skill

一打开浏览器， AI资讯目不暇接。

36kr 有深度报道， 量子位有快讯， 机器之心有技术分析。

一个个网站看过去， 光是打开和加载就花掉几分钟时间。

然后又浏览了半个小时， 对自己有用的消息就那么2-3条。

## 问题
- 信息源分散
  同一个新闻36kr发了， 量子位又发了一遍， 讲的都是同一件事情。
- 筛选效率低
  80%内容是噪音，20%才是有用的内容
- 整理耗时
  复制标题， 写摘要， 贴链接 又半小时

## AI 日报Skill
AI 咨询小助手
你只需要说 获取今天的AI新闻， 他就能自动去各大网站抓取最新AI资讯， 去掉重复内容， 然后生成一份精美的前端日报页面。

## Agent Browser 

1.png

它是一个浏览器自动化工具， 可以像人一样操作浏览器， 打开网页， 点击按钮和提取内容

Claude Code 是大脑
Agent Browser 是手和眼睛

只需要安转Agent Browser, cc 就拥有了操作浏览器的能力。

不需要你写任何操作浏览器的代码， cc 就可以调用Agent Browser 完成浏览器操作。

### 示例
agent-browser open https://juejin.cn/

## 两个阶段
- 数据获取
使用Agent Browser 分别从AI Base 和 36kr ,把抓取到的内容整理成JSON格式。
包含标题、摘要、链接和标签。

五条约束规则，为了获取数据的稳定性

- 第二阶段 展示页面生成
1. 赛博朋克风格的响应式AI日报页
2. 卡片式布局- 每张卡片展示一条资讯
3. 完整信息 
4. 响应式设计 支持移动端和桌面端
5. 日期与条数 显示当天日期和资讯总数
