## 创建会议纪要skill
启动claude code 
```
请帮我创建一个 skill，这个 skill 用于生成会议纪要
根据用户提供的文字稿，生成会议纪要，具体流程为：
先通读全文，去除停顿词等与会议无关内容
结构化输出：
会议基本信息（时间，地点，人物）
会议目标（目的 / 预期，背景信息）
会议内容（按主题划分，讨论了什么，观点是什么）
行动项（谁在何时需要做什么）
不确定的内容请留空
请将技能创建在当前工作目录的 /.claude/skills 文件夹中
```
会调用刚刚安装的skill creator 工具，读取创建skill的规范，开始生成会议纪要skill。

创建完后， 重启下claude code 
/skills 列出
@meeting_content.txt 总结会议纪要 
@ 添加上下文

成功调用会议纪要skill， 生成会议纪要。
