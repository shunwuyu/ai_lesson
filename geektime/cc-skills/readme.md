# Claude Code Skill 实战课

如果你已经受够了重复繁重的工作内耗，通过SKILL 交给AI.腾出时间做更有价值的事情。直接把我们的电脑变成可以跑各种AI技能的超级工作台。

## 聊聊几个场景的工作日常
1. 会议总结
开了一个小时的会议， 还需要花半个小时去整理会议纪要。
写清楚待办事项是什么， 谁负责， 截止日期是哪天？
每次都是相同的格式。 只要会议一结束， 就得再干一遍。
重复又麻烦。

2.竞品拆解
老板发过来一张竞品截图， 问， 你看看人家这个功能是怎么实现的？
你盯着这张竞品截图看了老半天， 心里明明知道人家做得好， 但是数不出好在哪里。
拆解不出具体的字段和逻辑。
3.编写需求文档
PRD 是产品需求文档， 用于描述产品的功能和非功能需求。
有固定的格式。
4.整理用户反馈
一打开用户后台， 映入眼帘的是一堆密密麻麻的用户反馈。
该提炼什么重点，
5. 数据分析
十几万行的数据， 怎么分析， 怎么写SQL
总是抓不住重点
做不出有用的分析

## 会议纪要

只需输入一段会议录音转成的文字稿，这个skill 就可以帮你生成一份内容完整的会议纪要。
待办事项、负责人、截止日期、风险点全部给你罗列的明明白白。

## 竞品拆解
上传一张竞品的截图，他能自动识别页面里的元素，接着反推出背后的业务逻辑

## 需求清洗
只需输入带用户评论的csv文件，它能够自动过滤掉没用的废话。精准提炼出核心需求。还能生成产品路线优化图。方便后续进行产品迭代。

## PRD 文档
你只需要说一句大白话， 做一个瑞幸积分的兑换功能。
它能主动帮拆解需求，直接生成一份完整详细的PRD文档。

## 可交互原型
只需要一句话，能生成一个能点，能操作，能交互的网络原型。
不需要你有任何的代码基础。

## 数据分析
上传一份业务数据， 他能自动读取字段，进行分析。直接输出趋势图、对比图，还有完整的业务结论。

## AI 日报
只需要提供几个信息源， 能自动获取、赛选、整理，输出一份结构化的AI日报。
自己看或汇报都特别使用。
