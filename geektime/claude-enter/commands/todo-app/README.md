# TODO 控制台应用

一个简单易学的 Node.js 控制台待办事项应用，使用 JSON 文件存储数据。

## 功能特性

- ✅ 添加待办事项
- 🗑️ 删除待办事项
- ✏️ 更新待办事项
- 🔄 切换完成状态
- 📋 查看所有待办事项
- 💾 JSON 文件持久化存储

## 项目结构

```
todo-app/
├── package.json          # 项目配置
├── README.md            # 本文件
├── src/
│   ├── index.js         # 主入口，用户界面
│   ├── todo.js          # 业务逻辑模块
│   ├── storage.js       # 数据存储模块
│   └── test.js          # 测试文件
└── data/
    └── todos.json       # 数据存储文件（自动创建）
```

## 快速开始

### 1. 运行应用

```bash
npm start
```

### 2. 运行测试

```bash
npm test
```

## 使用说明

启动后按菜单提示操作：

1. **查看所有待办事项** - 显示列表，包含完成状态
2. **添加待办事项** - 输入标题创建新待办
3. **删除待办事项** - 根据ID删除
4. **更新待办事项** - 根据ID修改标题
5. **标记完成/未完成** - 切换状态
0. **退出** - 关闭应用

- 📦 **模块化设计** - 代码分离，职责清晰
