- “如果函数本质上是对象，那为什么 typeof function() {} 不是 'object'，而是 'function' 呢？”

因为 JavaScript 为了方便开发者识别可调用对象，在 typeof 操作符中为函数专门加了一个特殊返回值 "function"。
从底层结构上讲，函数确实是对象，只是 typeof 对它开了一个“特例”。
