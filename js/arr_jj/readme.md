# 如何高效地创建、遍历与搜索数组？

数组（Array）是除了对象以外的最常用的数据集合结构。

- C++, java 静态编译型语言中
    数组只能存储同类型的数组，且数组的长度是固定的
    在内存上是一片连续的区域
    在定位、遍历上非常高效

- JS
    JavaScript 的数组更加灵活，不但可以存储不同类型的数组，而且数组的容量也可以灵活地扩展和收缩

    在实现数组上就有连续内存和哈希表两种结构。
