## 指令

以 / 开头的统一叫 斜杠命令，是 Claude Code 专属快捷指令。

### /init 

/init 就像让 Claude Code 先读一遍项目“说明书”，搞清技术栈、目录结构和代码规范后，再开始改代码

drumkit 通读项目
生成claude.md
