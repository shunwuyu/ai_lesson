- 善用AI是我们需要培养的最重要技能之一。
  - 不擅长的，  ai 生成令人沮丧的结果。
  - 擅长的， 使用AI工具后更强大

## AI novice & AI power user 两种体验

AI expert 借助它回答复杂问题
novice 回答简单问题， 就像在谷歌搜索一样
```
肯德基现在还有双层汉堡吗？
```

ai poser user
给复杂任务交给Ai 并给他一些思考时间

如果你想买辆车， 可以上传一些商业服务， 车辆规格，报价，保险

每款车各有哪些利弊取舍？通读全部内容，仔细思考后再作答。

AI会花上一些时间思考，并整理出一份详细报告。

另外

AI power user 懂得向AI提供合适的背景和上下文，
简短的提示往往无法提供足够的信息，导致AI无法正确理解问题。
```
写一份得体的自我评估报告发给我的上司
```
AI并不知道过去一年做了什么
非常笼统

AI power user 和AI共情
向AI上传大量信息

一张项目跟踪表， 展示你完成的工作量
最近的项目文档， 一些语音备忘录

```
写一份自我评估报告发给我的领导。以下是我完成的工作内容……
```

高手如何提示AI给出真诚反馈

需要AI 系统被训练为取悦用户

如果你提出一个带偏见的问题, 它通常会给出带偏见的内容
```
我有一个很棒的创业点子：移动扎染上门服务。你来点评分析一下这个项目？
```
AI 迎合我们
点子太棒了！这是一个新颖又有创意的生意项目。

熟练的Ai开发者 倾向于提出中性的问题
既不暗示期望的答案， 也不暗示不期望的答案

```
客观分析以下商业创意：上门移动扎染服务请依照评分标准作答。


```

- 写作

新手
```
写一篇关于黑莓手机的文章。
```
生成的内容有点像AI废话
AI power user
不会让系统直接写作，先让AI搭建大纲框架，随后进行评析、迭代优化，再拓展完善内容，

```
根据我的笔记，撰写一篇关于黑莓手机的博客文章大纲
```

有了大纲后， 给AI 反馈 喜欢或不喜欢的部分，多交流几次， 知道得到满意的提纲。
