# 预训练知识

我们可以通过阅读很多来学习， AI 也一样。它从互联网上大量文本中学习语言模式。

通过了解AI所阅读的内容，就可以预测它的行为方式。

- AI 能够解答各类不同主题的问题
```
我把手机掉进汤里了，该怎么办？
```

```
为什么猫咪会盯着墙壁看，仿佛它们看见了鬼魂一样?
```

- AI 模型基于海量数据源进行训练
通过对这些数据源的训练，模型形成**预训练**知识库
pre train

- Internet forums
- books
- encyclopedias
- news sites
- academic articles
训练AI 的大脑

- 出现频率高的内容， cooking, movies ... 
- 出现频率低的内容， 太空
- 没有的内容  私密数据

- 如果需要实时数据呢？
可以网络搜索
