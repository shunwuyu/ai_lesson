- 需求

嘴替贺卡，就是为了解决大家送生日祝福时**“想不出词儿”，但又想送上祝福的痛点。
现在的祝福都太干巴了，咱们要搞点好玩的。用户只需要无脑填几个关键词——比如朋友的名字、糗事或者爱好，甚至懒得想就直接点个“随机生成”，系统立马就能给凑出一篇特别逗的无厘头文案，还配好图。几十秒就能整出一张既个性又搞笑的卡片，直接复制发群里或者朋友圈，既显摆了幽默感，又把送祝福这个枯燥的任务给完成了。

- 想法到实现
  但在那之前，花点时间亲自搭建这个生日贺卡应用会大有帮助。这么做能让你更好地理解 “如何与 AI 沟通”，从而得到你想要的结果；这个实践过程还能培养你的直觉，让你知道 “如何引导 AI 生成内容”，以及哪怕是微小的调整，都可能带来截然不同的结果。
  等你对这些概念更熟悉之后，搭建自己的创意应用会变得更快、更简单，因为你已经知道如何把控整个过程了。

## Tell AI What you want 
  重要的不再是写代码， 而是告诉AI 你想要的结果。这就是Prompt。
  - 当给予精确的指令时，人工智能可以为你做很多事情。
  例子

  ``` 
  创建一个网页，帮助我撰写生日贺卡。GOAL
  当我提供一个人的名字、年龄和爱好时，它应该给我返回一条有趣的信息。 INPUT
  ```
  - 你想优化一下
  ```
  通过添加节日主题的标题和颜色，让它更美观。 OUTPUT
  ```
  - 再优化
  ```
  将贺卡显示在右侧，并使其看起来像生日贺卡的内页。  LAYOUT
  ```
  - 继续优化
  ```
  FEATURES 
  在页面顶部添加一个有趣的标题。并且不要替换旧的贺卡，而是保持它们可见，并将它们堆叠在新贺卡下方。
  ```

  #### building blocks of a good prompt 
  1.png
  GOAL + INPUT + OUTPUT + LAYOUT + FEATURES

  所以最后的prompt是

  ```
  创建一个网页来帮我写生日贺卡。

  当我输入一个人的姓名、年龄和爱好后，它应该返回一条有趣的消息。

  使用喜庆的标题和配色。

  把贺卡显示在右侧，并且让它看起来像生日贺卡的内页。

  在页面顶部添加一个有趣的标题。
  
  另外，不要替换旧的贺卡，而是保留它们的显示状态，把它们堆叠在新贺卡的下方。
  ```

## 给应用添加新特点
- 不设置三个输入框， 而希望是五个（爱好、星座、 故事、风格、 城市），更丰富
- 手气不错按钮 帮你填写所有字段
- 更新标题
- 添加按钮将信息加入剪贴版，方便发送给朋友
- 重新设计整个配色方案

通过决定你要添加哪些功能， 将ai 生成的应用变成你的应用。
- 包含五个明确标注的输入项：姓名、年龄、爱好、形容词、复数名词。
  详细： 不止是5个， 而是指定哪5个
- 添加一个“我感觉幸运”按钮，它会自动用随机词语填充空白处。提供一些有趣的预设词语选项。
- 让标题和副标题更精致一些。添加按钮，用于将卡片内容复制到剪贴板。
- 将配色主题设置为蓝色。
