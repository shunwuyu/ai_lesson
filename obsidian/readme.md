# Obsidian 搭建自己的知识库工具

小时候， 最向往有个自己的图书馆。
长大了，才发现读了再多的书， 记不住多少。
现在好了， 有了 Obsidian， 我可以把书籍、文章、笔记、代码等都放在里面。
我可以在 Obsidian 中搜索， 找到我需要的信息。

obsidian 我的专属图书馆，专属知识库。

## 知识库

就是把各种资料喂给AI，让它记住后随时能按你的提问精准回答的“智能大脑”。

## 安装

- Qwen2.5-Omni
Qwen2.5-Omni就是阿里推出的一个"全能型"AI模型，虽然只有70亿参数，但能同时看懂文字、图片、音频和视频，还能实时用语音跟你对话，就像一个长了眼睛耳朵嘴巴的聪明小助手。2025年3月27日由阿里巴巴发布并开源，采用 Apache 2.0 许可证，开发者和企业可以免费下载商用。
  - https://lmstudio.ai/ 下载lmstudio.ai
